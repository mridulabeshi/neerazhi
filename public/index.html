How it works

1

Sensors collect data

Ultrasonic water-level sensors in manholes send readings via MQTT every 2 seconds.

2

Server processes

Node.js backend calculates risk scores, runs path-finding, and pushes updates via WebSocket.

3

Dashboard renders

Canvas-based network map renders colour-coded nodes and the optimal diversion path in real time.

4

Claude analyses

On demand, the Anthropic API receives a full sensor snapshot and returns a practical flood assessment.
